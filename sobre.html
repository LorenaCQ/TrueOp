Objetivo

Criar uma rede de pessoas com deficiência, promover a inclusão digital e permitir a avaliação de outras aplicaçõesde empresas de terceiros.


Como Funciona?

Empresas de diversos ramos disponibilização seus sites para que pessoas com deficiência possam avalia-los. As empresas podem oferecer recompensas para o avaliador.


Recomendação Inteligente

A Ref.IA analisa os dados de acesso e cadastrados pelo usuário para recomendar sites acessíveis. Ela também recomenda os sites para determinados avaliadores que melhor se encaixam no público alvo do site, tudo isso de uma maneira autônoma. Além de recomendar sites para avaliação e separar os dados para pesquisas acadêmicas. Estes dados são disponibilizados em bases de dados públicas.
